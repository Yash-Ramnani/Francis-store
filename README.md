# Francis Store

Francis Store is a responsive e-commerce website showcasing clothing and daily essentials.

## Features
- Fully responsive design using HTML, CSS, Bootstrap, and JS
- Clean UI and smooth navigation
- Product gallery with images
- Cart functionality

## Folder Structure
css/ -> Stylesheets
js/ -> JavaScript files
images/ -> Product and UI images


## How to run
1. Clone or download this repo.
2. Open `FRancis.html` in your browser.
